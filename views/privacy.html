Privacy Policy

The Cards and Castles Discord Bot gets the data for it's content lookup from the spreadsheet you give it access to.

This data is extracted and stored as part of the Project Code.

The stored data may be visible to the public as part of the Project Code.

The stored data should be purely informational data about the Storm Wars Game and should not contain any personal information at all. Please do not enter any personal information into the spreadsheet the Bot is granted access to.
